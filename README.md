## Studies on Artificial Inteligence Development

This repo contains two projects that I developed during my Bachelor's classes on Artificial Inteligence.

### EP1 - MLP
This is the first project, it involved the devolpment of a Multilayer Perceptron Network that identifies handwritten letters.
The core idea of this project was to get a feel of all the steps in MLP system, so it was build using as few libraries as possible. 
Includes all the steps for an MLP implementation, such as forward/back propagation and training, all from scratch.
With a stack of pure Python and NumPy.

### EP2 - CNN
This second project was a follow up to the latter. The goal is the same: identify handwritten letters from the same dataset, but now using 
a Convolutional Neural Network, and we were allowed to use more libraries and tools to help on development.
Developed as a continuation of the MLP project, this project implements a Convolutional Neural Network (CNN) using the same dataset
Stack: Python, NumPy, Matplotlib, Scikit-learn, TensorFlow
